Сайт с параллакс эффектом

Параллакс-эффект — это видимое смещение или изменение положения объекта, когда вы смотрите на него с разных точек зрения (или когда вы сами перемещаетесь относительно объекта, а фон остается неподвижным). анимированны все элементы на сайте

Сайт запускается через любую среду разработки. Важно запускать имеено html файл в среде разработки

Файл normalize.css (или Normalize.css) — это небольшой CSS-файл, который отвечает за установление базовой кроссбраузерной консистентности стилей HTML-элементов, не сбрасывая их полностью, а приводя к единообразию.
Файл style.css — это, пожалуй, самый основной и универсальный файл, который отвечает за внешний вид и оформление всего вашего веб-сайта.
Файл index.html — это фундамент и основной входной файл любого веб-сайта или веб-приложения. Его можно сравнить с каркасом дома или оглавлением книги: он определяет структуру, содержание и связи с другими частями сайта.
Файл script.js — это файл, который содержит код на языке JavaScript и отвечает за всю интерактивность и динамическое поведение вашего веб-сайта.
