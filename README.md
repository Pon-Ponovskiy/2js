# Сайт с параллакс эффектом

Параллакс-эффект — это видимое смещение или изменение положения объекта, когда вы смотрите на него с разных точек зрения (или когда вы сами перемещаетесь относительно объекта, а фон остается неподвижным). анимированны все элементы на сайте

## Как запустить проект

Сайт запускается через любую среду разработки. Важно запускать имеено html файл в среде разработки

## Описание файлов

Файл normalize.css (или Normalize.css) — это небольшой CSS-файл, который отвечает за установление базовой кроссбраузерной консистентности стилей HTML-элементов, не сбрасывая их полностью, а приводя к единообразию.
Файл style.css — это, пожалуй, самый основной и универсальный файл, который отвечает за внешний вид и оформление всего вашего веб-сайта.
Файл index.html — это фундамент и основной входной файл любого веб-сайта или веб-приложения. Его можно сравнить с каркасом дома или оглавлением книги: он определяет структуру, содержание и связи с другими частями сайта.
Файл script.js — это файл, который содержит код на языке JavaScript и отвечает за всю интерактивность и динамическое поведение вашего веб-сайта.

## Переменные проекта

Переменные parallax__сontent , clouds , mountains , human хранят часть структуры сайта, которая принадлежит тому или иному изображению или контенту текста параллакса:
```js
        let parallax__content = document.querySelector('.parallax__content')
        let clouds = document.querySelector('.images__clouds')
        let mountains = document.querySelector('.images__mountains')
        let human = document.querySelector('.images__human')
```

  
Константы forClouds , forMountains , forHuman хранят динамичность анимации (чем больше значение, тем меньше будет динамичность того или иного объекта), a animationSpeed - скорость анимации (чем больше, тем быстрее будут перемещаться объекты):
```js
const forClouds = 48
const forMountains - 20
const forHuman = 10
const animationSpeed - 0.05
```

## Описание функций

Функция setParalaxAnimation() `описывает работу анимации при движении мыши 
requestAnimationFrame -`при скроле` 
`requestAnimationFrame() - `метод для оптимизации и плавности анимации`

Слушатель после setParalaxAnimation() `определяет положение мыши на экране и благодаря этому задействует анимацию.`
requestAnimationFrame `вызывается в колбэк функции, которая срабатывает при достижении порогов, далее переменная observer (наблюдатель) присваивается элементу, который является начальной точкой анимации при прокрутке.`
